# 🏥 Sistema de Gestão — Clínica Saúde+

Sistema completo de análise de dados e automação 
para clínicas privadas em Maputo, Moçambique.

## 📊 O que o sistema faz

- Dashboard interactivo em tempo real
- Análise de receita por médico e por mês
- Controlo de pagamentos em atraso
- Relatório automático enviado por email mensalmente

## 🛠️ Ferramentas utilizadas

- **Google Sheets** — base de dados
- **Looker Studio** — dashboard e visualização
- **Make.com** — automação de relatórios

## 📁 Estrutura dos dados

| Coluna | Descrição |
|--------|-----------|
| Data | Data da consulta |
| Paciente | Nome do paciente |
| Médico | Nome do médico |
| Tipo de Consulta | Geral, Pediatria, Cardiologia |
| Valor (MT) | Valor em Meticais |
| Estado Pagamento | Pago / Em atraso |

## ⚙️ Automação

O sistema usa Make.com para:
1. Ler os dados do Google Sheets automaticamente
2. Agregar todas as consultas num resumo
3. Enviar relatório por email mensalmente
